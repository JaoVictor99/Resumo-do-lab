# Resumo-do-lab

O Microsoft Azure é uma plataforma de computação em nuvem da Microsoft que oferece uma ampla variedade de serviços, como infraestrutura, plataforma e software (IaaS, PaaS e SaaS). Seu objetivo é permitir que empresas e desenvolvedores criem, implantem e gerenciem aplicações com segurança, escalabilidade e alta disponibilidade.

A arquitetura do Azure é baseada em regiões e zonas de disponibilidade, que garantem desempenho e confiabilidade global. A plataforma fornece serviços essenciais de computação, armazenamento, banco de dados, redes, inteligência artificial, segurança e Internet das Coisas (IoT).

O gerenciamento dos recursos pode ser feito pelo Azure Portal, Azure CLI, PowerShell e pelo Azure Resource Manager, que organiza e automatiza a administração dos serviços. Em relação à segurança, o Azure conta com ferramentas como Microsoft Defender for Cloud, Azure Policy, Key Vault e Microsoft Entra ID, que protegem dados, identidades e aplicações.

O modelo de pagamento sob demanda permite flexibilidade e controle de custos, ajustando os recursos conforme a necessidade. A certificação AZ-900 representa o nível fundamental de conhecimento sobre o Azure, validando os conceitos básicos de computação em nuvem e os principais serviços oferecidos pela plataforma.
